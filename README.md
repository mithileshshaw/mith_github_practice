4 types of status :-
1. untracked -- new file that git doesn't yet track
2. modified -- changed
3. staged -- file is ready to committed
4. unmodified -- unchanged


ADD and Commit :-
1. add -- adds new or changed files in your working directory to the Git staging area
        git add <-filename-> or git add . -- to add multiple file
2. commit - it is the record of change
        git commit -m "some message
